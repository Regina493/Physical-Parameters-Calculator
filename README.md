# Physical Parameters Calculator 📊

Простое и надежное консольное приложение на C++ для расчета и анализа физических показателей пользователя.

## ✨ Особенности / Features:
* *Валидация данных:* Скрипт защищен от ошибок ввода (например, если ввести буквы вместо цифр).
* *Интерактивность:* Удобный диалог с пользователем и поддержка команд выхода.
* *Точность:* Использование типа double для корректных математических расчетов.

## 🚀 Как запустить / How to run:
1. Скомпилируйте файл (Compile):
   g++ main.cpp -o calculator
2. Запустите программу (Run):
   ./calculator.exe
